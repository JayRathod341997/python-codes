# Project 08 — Payroll CSV to JSON Processing

## The Real-Life Problem

An HR system needs to:
- Read employee data from a CSV file (emp_id, name, salary)
- Compute deductions (PF 12%, TDS based on slab)
- Generate payslips
- Output to CSV (for accounting) and JSON (for API)

This teaches file I/O, CSV/JSON handling, exception handling, and pathlib for directory management.

## Domain Context

**Industry**: HR / Payroll / Finance
**Tools**: payroll software, accounting systems, APIs
**Why Python**: Data processing and file generation are Python's strength

## Concepts

CSV module, JSON module, pathlib, with statement, exception handling, dict manipulation, list comprehensions
